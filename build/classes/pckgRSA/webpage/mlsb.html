Modified LSB

The original LSB method can be modified and improved in several ways. In our application we are implementing a modified version of LSB by altering the order in which pixels are chosen from the cover image for embedding the message bits. In the original version pixels are chosen in a linear fashion, i.e, from left to right in each row and from first to last row in increasing order.When the bit planes of the image are examined by an observer, the location of the embedding will be easily detected. Since the pixels of the image are organized in the same manner as the original image, the observer will be able to decode the image without any extra effort. The Spiral Embedding is novel embedding technique designed to resist a visual attack. The Spiral Embedding orders the pixels of the image in a spiral pattern to prevent the embedding from being as easily decoded.
