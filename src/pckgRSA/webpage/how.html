How it's done

In this section we will provide you with a simple guide on how to use the application. The various components of the application and each of their purpose are given below:

  1. Message Pane : The message is typed here or a text file can be displayed.
  2. Clear : This button is used to clear the text in the message pane.
  3. Save : This button is used to save the text displayed in the message pane.
  4. Open : This button is used to open a text document and display it in the message pane.
  5. Encryption Algorithm : This radio buttons are used to select the algoritm used for encryption or decryption.
  6. Steganoography Algorithm : This radio buttons are used to select the algoritm used for hiding or retrieve message from image.
  7. Generate Key : This button is used to generate a key for the selected algorithm to be used for encryption-decryption.
  8. Choose Key : This button is used to choose a previously generated key for the selected algorithm to be used for encryption-decryption.
  9. Encrypt : This button is used to encrypt the message in the message pane and display the corresponding cypher text.
  10. Decrypt : This button is used to decrypt the cypher text in the message pane and display the original message.
  11. Hide : This button is used to hide the text in the messsage pane within the image.
  12. Retrieve : This button is used to retrieve the hidden text from the stego image and display it in the message pane.
  13. Send/Receive : This button is used to send/receive the key, encrypted messages or stego images.
  14. Image Pane : The cover image/ stego image gets displayed here.
  15. Choose Image : This button is used to choose an image.
  16. Analyze : This button is used to analyze whether the message will fit in the cover image.
  17. Notification Pane : Various notifications or suggestions are displayed here.
  18. Help : This button is used to display the help section.

You can do the following with your application:

  • Encrypt a message
  • Decrypt an encrypted message
  • Hide a message in an image
  • Retrieve hidden message from an image
  • Encrypt a message and then hide it in an image
  • Retrieve encrypted message from an image and then decrypt it
  • Use multiple encryption
  • Use Digital Envelope technique
  • Try various combinations of above techniques

Encrypting a message

Steps:

  1. Write a message in the message pane or open a text document.
  2. Choose the encryption algorithm you want to use.
  3. Generate or Choose a key/s.
  4. Hit the "Encrypt" button.
  5. Save the encrypted text.
  6. Send the key(public) and the encrypted message to the receiver(incase of Symmetric key cryptography it is not advised to send the key through email).

Decrypting an encrypted message

Steps:

  1. Receive the encrypted message and key(public) from sender.
  2. Open the encrypted message.
  3. Choose the same encryption algorithm as used by sender.
  4. Choose the key from the sender.
  5. Hit the "Decrypt button".
  6. Save the original message.

Hiding a message in an image

Steps:

  1. Write a message in the message pane or open a text document.
  2. Choose the steganography algorithm you want to use.
  3. Hit the "Choose Image" button to choose a cover image.
  4. Hit the "Hide" button and save the stego image.
  5. Send the stego image to the receiver.

Retrieveing hidden message from an image

Steps:

  1. Receive the stego image sent by sender.
  2. Choose the image sent by the sender by hitting the "Choose Image" button.
  3. Choose the same steganography algorithm as used by the sender.
  4. Hit the "Retrieve" button and the original message is displayed in the message pane.
  5. Save the original message.

Encrypt + Hide

Steps:

  1. Write a message in the message pane or open a text document.
  2. Choose the encryption algorithm you want to use.
  3. Generate or Choose a key/s.
  4. Hit the "Encrypt" button.
  5. Choose the steganography algorithm you want to use.
  6. Hit the "Choose Image" button to choose a cover image.
  7. Hit the "Hide" button and save the stego image.
  8. Send the key(public) and the stego image to the receiver.

Retrieve + Decrypt

Steps:

  1. Receive the stego image and the key(public) sent by sender.
  2. Choose the image sent by the sender by hitting the "Choose Image" button.
  3. Choose the steganography algorithm used by the sender.
  4. Hit the "Retrieve" button and the encrypted message is displayed in the message pane.
  5. Receive the key from sender.
  6. Choose the same encryption algorithm as used by sender.
  7. Choose the key from the sender.
  8. Hit the "Decrypt button".
  9. Save the original message.

Multiple-Encryption

Steps:

  1. Write a message in the message pane or open a text document.
  2. Choose the encryption algorithm you want to use.
  3. Generate or Choose a key/s.
  4. Hit the "Encrypt" button.
  5. Save the encrypted text.
  6. Send the key(public) to the receiver.
  7. Repeat steps 2 to 5 as many times as you want.
  8. Send the encrypted message to the receiver.

Multiple-Decryption

Steps:

  1. Receive the encrypted message and keys(public) from sender.
  2. Open the encrypted message.
  3. Choose the encryption algorithms in the reverse of the order as used by sender.
  4. Choose the keys from the sender in reverse of order in which they were sent.
  5. Hit the "Decrypt button".
  6. Repeat steps 2 to 5 as many times as the number of keys.
  7. Save the original message.

Digital Envelope Technique

Encryption

Steps:

  1. Choose the RSA encryption algorithm.
  2. Generate the keys.
  3. Choose the AES encryption algorithm.
  4. Generate a key.
  5. Open the AES key in the message pane
  6. Choose the RSA encryption algorithm.
  7. Choose the same RSA keys
  8. Hit the "Encrypt" button.
  9. Save the encrypted key.
  10. Send the encrypted AES key and the RSA public key to the receiver.
  11. Write a message in the message pane or open a text document.
  12. Choose AES encryption algorithm
  13. Choose the same AES key
  14. Hit the "Encrypt" button.
  15. Send the encrypted message to the receiver.

Decryption

Steps:

  1. Choose the RSA encryption algorithm.
  2. Choose the RSA key sent by the sender.
  3. Choose the AES encryption algorithm.
  4. Open the AES key in the message pane
  5. Hit the "Decrypt" button.
  6. Save the decrypted key.
  7. Choose AES encryption algorithm.
  8. Choose the saved decrypted AES key.
  9. Open the encrypted message in the message pane.
  10. Hit the "Decrypt" button.
  11. Save the original message.
